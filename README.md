# Medolin Shajeena — Interactive Resume

Single-file portfolio/resume site. Material You design system with:
- Dynamic color: pick a seed swatch in the top bar to re-theme the whole site
- Light/dark mode (respects OS preference)
- Runtime WCAG AA contrast guard — every generated palette is verified >= 4.5:1
- Live design-system specimens, Human+AI workflow explainer
- Fully keyboard-accessible: skip link, roving-tabindex tabs, aria-live regions

## Deploy
No build step. `index.html` is the entire site.

**GitHub Pages:** push this folder to a repo named `<username>.github.io`
(or any repo → Settings → Pages → Deploy from branch → main /root).

**Netlify:** drag this folder onto https://app.netlify.com/drop
